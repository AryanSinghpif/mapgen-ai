What it does

Upload a CSV or Excel file with district-level data. mapgen resolves name variants, detects the state, crops the boundary file, and renders a publication-ready map.

Honest by design

Missing districts render as gray hatching — never as zero. Every map carries a source line and boundary vintage year.

Export formats

PNG · JPG · SVG · Interactive HTML · GeoJSON · Standalone Python script that regenerates the exact map.
